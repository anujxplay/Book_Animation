# 📖 Book Animation

A stylish and interactive **3D Book Flip Animation** built using **HTML, CSS, and JavaScript**.  
This project creates a realistic page-turning effect with smooth animations, making it perfect for portfolios, landing pages, digital magazines, storybooks, or UI experiments.

---

## 🚀 Live Demo

🌐 [View Live Demo](https://anujxplay.github.io/Book_Animation/)

📂 [GitHub Repository](https://github.com/anujxplay/Book_Animation)

## ✨ Features

- 📄 Realistic page flip animation
- 🎨 Clean and modern UI
- ⚡ Smooth CSS transitions
- 📱 Responsive design
- 🖱️ Interactive navigation
- 💻 Pure HTML, CSS & JavaScript
